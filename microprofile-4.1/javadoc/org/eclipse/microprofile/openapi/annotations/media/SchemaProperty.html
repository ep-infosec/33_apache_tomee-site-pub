org.eclipse.microprofile.openapi.annotations.media

Annotation Type SchemaProperty


  • @Retention(value=RUNTIME)
 @Inherited
public @interface SchemaProperty
    The SchemaProperty Object allows the definition of input and output data types nested within the properties attribute of a Schema annotation. These types can be objects, but also primitives and arrays. This object is an extended subset of the JSON Schema Specification Wright Draft 00.
    Since:
    2.0
    See Also:
    OpenAPI Specification Schema Object
