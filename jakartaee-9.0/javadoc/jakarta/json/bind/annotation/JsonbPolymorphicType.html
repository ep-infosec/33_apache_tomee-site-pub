jakarta.json.bind.annotation

Annotation Type JsonbPolymorphicType


  • @JsonbAnnotation
 @Retention(value=RUNTIME)
 @Target(value={ANNOTATION_TYPE,TYPE})
public @interface JsonbPolymorphicType
    Configuration annotation of the polymorphic type handling.
    This annotation is required on the most common parent of all classes when polymorphism will be applied. 
    
 // Example
  @JsonbPolymorphicType(key = "@key")
 interface Animal {}

 class Dog implements Animal {}
 class Cat implements Animal {}
    This annotation is tightly bound to JsonbSubtype. It is recommended to use JsonbSubtype annotations to specify all the possible classes and their aliases.
