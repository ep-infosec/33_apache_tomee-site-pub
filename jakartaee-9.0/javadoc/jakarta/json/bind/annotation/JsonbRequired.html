jakarta.json.bind.annotation

Annotation Type JsonbRequired


  • @JsonbAnnotation
 @Retention(value=RUNTIME)
 @Target(value={ANNOTATION_TYPE,METHOD,PARAMETER})
public @interface JsonbRequired
    Specifies which parameters of the JsonbCreator are required.
    There are two possible values for this parameter: false or true. If set to false, parameters are marked as optional and will not be required in the JSON document. If set to true, however, parameters will be required (default behavior).
    Annotation can be specified on method or constructor annotated also by JsonbCreator and affects all the creator parameters. It is also possible to use this annotation on the creator parameter.
    Since:
    JSON Binding 2.1
