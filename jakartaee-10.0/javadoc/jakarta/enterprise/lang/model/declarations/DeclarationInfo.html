jakarta.enterprise.lang.model.declarations

Interface DeclarationInfo

  • All Superinterfaces:
    AnnotationTarget
    All Known Subinterfaces:
    ClassInfo, FieldInfo, MethodInfo, PackageInfo, ParameterInfo, RecordComponentInfo

    public interface DeclarationInfo
extends AnnotationTarget
    A declaration is an element of a program source code. Declarations are:
    • packages
    • classes, including interfaces, enums, annotations, and records
    • fields
    • methods, including constructors
    • method parameters, including constructor parameters
    Since:
    4.0
