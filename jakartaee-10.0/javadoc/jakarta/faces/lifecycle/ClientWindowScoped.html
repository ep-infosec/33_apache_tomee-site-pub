jakarta.faces.lifecycle

Annotation Type ClientWindowScoped


  • @NormalScope(passivating=true)
 @Inherited
 @Documented
 @Target(value={TYPE,FIELD,METHOD})
 @Retention(value=RUNTIME)
public @interface ClientWindowScoped
    ClientWindowScoped is a CDI scope that causes the runtime to consider classes with this annotation to be in the scope of the current ClientWindow. The implementation must provide an implementation of jakarta.enterprise.inject.spi.Extension that implements the semantics such that beans with this annotation are created lazily, when referenced and accessed.
    Since:
    4.0
